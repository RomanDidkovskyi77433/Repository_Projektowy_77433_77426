# 📚 Biblioteka — Projekt Semestralny

Aplikacja webowa do zarządzania katalogiem książek.

## Technologie
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js + Express
- **Baza danych:** JSON (lowdb)
- **Hosting:** Railway

## Uruchomienie lokalne

```bash
npm install
npm start
```

Aplikacja dostępna pod: http://localhost:3000

## Funkcje
- Przeglądanie katalogu książek
- Dodawanie nowych książek (tryb edycji)
- Edytowanie istniejących książek
- Usuwanie książek
- Wyszukiwarka po tytule i autorze
- Szczegóły książki w oknie modalnym
